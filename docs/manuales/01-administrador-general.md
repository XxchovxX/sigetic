# Manual De Usuario - Administrador General

## Objetivo Del Rol

El administrador general tiene control total sobre SIGETIC. Puede gestionar usuarios, dependencias, funcionarios, activos TIC, impresoras, consumibles, tickets, reportes, analítica, auditoría y configuración.

## Módulos Disponibles

- Dashboard
- Inventario TIC
- Hojas de vida
- Impresoras
- Consumibles
- Mesa de ayuda
- Ayuda
- Dependencias
- Funcionarios
- Reportes
- Analítica
- Auditoría
- Configuración

## Actividades Principales

### Crear Usuarios

1. Ingrese a **Configuración**.
2. Diligencie nombre completo, correo, contraseña temporal y rol.
3. Presione **Crear usuario**.
4. Entregue la contraseña temporal al usuario por un canal seguro.

### Cambiar Contraseña De Un Usuario

1. Ingrese a **Configuración**.
2. Busque el usuario en la tabla de accesos registrados.
3. Escriba la nueva contraseña.
4. Presione **Cambiar**.
5. Informe al usuario que debe ingresar con la nueva clave.

### Administrar Dependencias Y Funcionarios

1. Ingrese a **Dependencias** para crear o actualizar áreas institucionales.
2. Ingrese a **Funcionarios** para registrar personas responsables o asignadas.
3. Use nombres completos y dependencias correctas para que reportes y hojas de vida salgan limpios.

### Revisar Auditoría

1. Ingrese a **Auditoría**.
2. Filtre por módulo, acción, usuario o fecha.
3. Revise el detalle de la operación.
4. Use esta información para seguimiento interno y control.

### Gestionar Activos, Impresoras Y Consumibles

El administrador puede crear, editar, consultar, exportar y dar seguimiento a todos los activos tecnológicos, impresoras y consumibles.

## Buenas Prácticas Del Administrador

- Cree usuarios solo con el rol que corresponde a su función.
- Inactive usuarios que ya no deben ingresar.
- No comparta la cuenta administradora.
- Revise auditoría cuando se detecten cambios no esperados.
- Mantenga actualizadas dependencias y funcionarios antes de registrar activos.

## Restricciones Recomendadas

Aunque el rol puede hacer todo, se recomienda que el administrador general no use su cuenta para operación diaria de mantenimiento. Para eso deben usarse cuentas técnicas como **Administrador TIC** o **Auxiliar de Sistemas**.
