# Manual De Usuario - Secretario Administrativo Financiero

## Objetivo Del Rol

El Secretario Administrativo Financiero consulta la información institucional, revisa indicadores, genera reportes, crea tickets y puede actualizar consumibles relacionados con la gestión administrativa y financiera.

## Módulos Disponibles

- Dashboard
- Inventario TIC
- Hojas de vida
- Impresoras
- Consumibles
- Mesa de ayuda
- Ayuda
- Reportes
- Analítica

## Actividades Principales

### Revisar Dashboard

1. Ingrese al sistema.
2. Consulte indicadores de equipos, impresoras, tickets y stock.
3. Use esta vista para seguimiento general.

### Consultar Inventario Y Hojas De Vida

1. Ingrese a **Inventario TIC** o **Hojas de vida**.
2. Busque por código, serial, dependencia o funcionario.
3. Abra el detalle cuando necesite revisar información.
4. Exporte cuando se requiera soporte documental.

### Gestionar Consumibles

1. Ingrese a **Consumibles**.
2. Revise existencias, costos y movimientos.
3. Registre entradas, salidas o ajustes permitidos.
4. Verifique que el sistema descuente automáticamente cuando se registra consumo.

### Crear Ticket

1. Ingrese a **Mesa de ayuda**.
2. Presione **Crear ticket**.
3. Describa la solicitud con claridad.
4. Guarde y haga seguimiento al estado.

### Generar Reportes

1. Ingrese a **Reportes**.
2. Seleccione el tipo de reporte.
3. Defina periodo o filtros.
4. Exporte PDF o Excel.

## Restricciones Del Rol

Este rol no administra usuarios, dependencias ni funcionarios. Tampoco registra o edita activos técnicos como responsable operativo.

## Buenas Prácticas

- Use reportes mensuales para seguimiento administrativo.
- Revise alertas de stock antes de autorizar nuevas compras.
- Verifique consumos por dependencia.
- Use mesa de ayuda para dejar trazabilidad de solicitudes.
