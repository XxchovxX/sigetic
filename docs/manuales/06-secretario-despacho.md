# Manual De Usuario - Secretario De Despacho

## Objetivo Del Rol

El Secretario de Despacho puede consultar el dashboard institucional y crear solicitudes de soporte para su dependencia.

## Módulos Disponibles

- Dashboard
- Mesa de ayuda
- Ayuda

## Actividades Principales

### Consultar Dashboard

1. Ingrese al sistema.
2. Revise indicadores generales.
3. Use la información para seguimiento de solicitudes y estado tecnológico institucional.

### Crear Ticket

1. Ingrese a **Mesa de ayuda**.
2. Presione **Crear ticket**.
3. Seleccione el tipo de solicitud.
4. Describa el problema o necesidad.
5. Guarde el ticket.

### Hacer Seguimiento

1. Ingrese a **Mesa de ayuda**.
2. Revise el estado de las solicitudes.
3. Consulte observaciones registradas por el área TIC.

## Restricciones Del Rol

No administra inventario, impresoras, consumibles, usuarios, reportes técnicos ni auditoría.

## Buenas Prácticas

- Cree una solicitud por cada caso.
- Describa ubicación, equipo afectado y urgencia.
- Evite crear tickets duplicados para el mismo caso.
