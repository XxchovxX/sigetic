# Manual De Usuario - Consulta / Control Interno

## Objetivo Del Rol

El rol Consulta / Control Interno permite revisar información institucional, reportes y analítica sin modificar datos operativos ni administrar usuarios.

## Módulos Disponibles

- Dashboard
- Inventario TIC
- Hojas de vida
- Impresoras
- Mesa de ayuda
- Ayuda
- Reportes
- Analítica

## Actividades Principales

### Consultar Información

1. Ingrese al módulo requerido.
2. Use buscadores o filtros.
3. Abra detalles cuando necesite revisar información específica.

### Revisar Reportes

1. Ingrese a **Reportes**.
2. Seleccione el reporte requerido.
3. Exporte PDF o Excel si se necesita soporte documental.

### Revisar Analítica

1. Ingrese a **Analítica**.
2. Consulte indicadores de tickets, consumibles, mantenimientos y atención.
3. Use la información para seguimiento y control.

### Consultar Tickets

1. Ingrese a **Mesa de ayuda**.
2. Revise tickets y estados.
3. Use la información como trazabilidad de atención.

## Restricciones Del Rol

No puede crear, editar o eliminar activos técnicos, impresoras, consumibles, usuarios, dependencias o funcionarios. Tampoco tiene acceso a configuración.

## Buenas Prácticas

- Use filtros antes de exportar información.
- Contraste reportes con periodos definidos.
- Solicite al administrador cualquier corrección de datos detectada.
