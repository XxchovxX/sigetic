# Manual De Usuario - Funcionario

## Objetivo Del Rol

El funcionario usa SIGETIC para crear solicitudes de soporte y consultar el avance de sus propios tickets.

## Módulos Disponibles

- Mesa de ayuda
- Ayuda

## Crear Un Ticket

1. Ingrese al sistema.
2. Abra **Mesa de ayuda**.
3. Presione **Crear ticket**.
4. Diligencie asunto, tipo de solicitud, dependencia y descripción.
5. En la descripción indique qué ocurre, dónde ocurre y desde cuándo ocurre.
6. Guarde el ticket.

## Consultar El Estado De Un Ticket

1. Ingrese a **Mesa de ayuda**.
2. Revise el listado de sus solicitudes.
3. Abra el ticket para ver observaciones, estado y avance.

## Estados Más Comunes

- **Abierto:** solicitud recibida.
- **En proceso:** el área TIC está atendiendo el caso.
- **En verificación:** se está validando la solución.
- **Cerrado:** caso finalizado.

## Encuesta De Satisfacción

Cuando un ticket sea atendido, el sistema puede solicitar calificación sobre tiempo de respuesta, atención, amabilidad y solución recibida. Esta información ayuda a mejorar el servicio.

## Buenas Prácticas

- No cree varios tickets para el mismo problema.
- Describa el equipo, ubicación y error.
- Adjunte o escriba evidencia si el sistema lo permite.
- Revise el estado antes de solicitar una actualización por otro medio.
