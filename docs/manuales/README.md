## Criterios Generales

- Cada usuario debe ingresar con su cuenta asignada.
- Las contraseñas temporales deben cambiarse después de la entrega inicial.
- Las acciones relevantes quedan registradas en auditoría cuando el módulo lo soporta.
- La firma escaneada o digitada se usa como firma electrónica interna, no como firma digital certificada.
- La información exportada en PDF o Excel debe tratarse como información institucional.
